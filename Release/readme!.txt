      Генератор ролевых элементов для TES 5

Этот генератор, несмотря на аскетичный дизайн, позволит вам в секунды
создать описание ролевой составляющей своего персонажа. Генератор
представляет сотни комбинаций - для неискушенных это станет способом
взглянуть на мир TES иначе, для харкорного ролеплейщика она станет
источников новых идей. Более того, генератор ориентирован на любую часть
из серии TES: от Arena до Skyrim.
Советую использовать вместе с игрой Мод на альтернативное начало, для более правдоподобного отогрыша.

-----------------------------------------------------------------------------------------------
Инструкция:                                                                                    |
1) для запуска программы нажмите "ЗАПУСК".                                                     |
                                                                                               |
2) Если вы хотите повторить генерацию, введите "y" и нажмите "enter".                          |
                                                                                               |
3) Если вы хотите посмотреть результат генерации или поделиться с друзьями, то                 |
все результаты генерации хранятся в текстовом файле "randomfile.txt" после закрытия программы, |
однако они удаляются после повторного запуска программы.                                       |
